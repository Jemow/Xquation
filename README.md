# Xquation

## Overview

**Xquation** is a math-themed top-down arena shooter created in under one week for Chili Code Jam #4. The game blends fast-paced roguelike combat with dynamic mathematical gameplay.

Players fight waves of enemies using standard projectiles and a unique laser beam mechanic. The twist: the laser beam follows a mathematical function that the player builds throughout the run. By combining functions and variables, players can create unpredictable and powerful beam patterns.

Between waves, players interact with the **Function Lab**, where they modify their current function using arithmetic operations and compositions. This allows for emergent gameplay and experimentation, especially with the addition of a time-based variable that introduces chaotic motion.

[Play on itch.io](https://jemow.itch.io/xquation)

## Features

* Top-down arena shooter gameplay
* Dynamic function-based laser system
* Function Lab to build and modify equations during runs
* Roguelike progression with increasing difficulty waves
* Time variable (t) that oscillates and affects beam behavior
* Mathematical operations: Add, Subtract, Multiply, Divide, Compose
* Variety of functions: trigonometric, inverse, logarithmic, and more
* Encourages experimentation with math-driven mechanics

## Controls

* Move — WASD / Arrow Keys
* Fire Projectile — Left Mouse Button (LMB)
* Laser Beam — Right Mouse Button (RMB)
* Switch Projectile Type — F
